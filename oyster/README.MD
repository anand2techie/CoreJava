# Oyster
  
A simple Oyster (London public transport) implementation.

#Environmental setting
Set your Java to 8
  
## Build & Run
Build with maven (`mvn clean package`). Then run with Java: `java -jar oyster-0.0.1-SNAPSHOT.jar`
